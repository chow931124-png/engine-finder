# 🧠 Serenity式 AI选股辩论系统

## 简介

复刻 Serenity 的 AI 魔鬼代言人选股模式。对任意 A 股标的，从**技术壁垒、财务基本面、估值安全边际、催化剂时机**四个维度进行交叉辩论。每个维度的 AI 扮演空头，全力攻击该股——只有扛住所有攻击的标的才值得建仓。

## 使用方式

在 Claude Code 对话中说：

```
/workflow stock-debate 600522
```

或：

```
帮我用Serenity模式辩论一下中天科技(600522)
```

## 辩论流程

```
数据收集 (WebSearch + akshare MCP)
    │
    ├── 🔬 技术壁垒魔鬼代言人 ──→ 评分 + 致命攻击
    ├── 📊 财务基本面魔鬼代言人 ──→ 评分 + 致命攻击
    ├── 💰 估值魔鬼代言人 ──→ 评分 + 致命攻击
    └── ⚡ 催化剂时机魔鬼代言人 ──→ 评分 + 致命攻击
                    │
                    ▼
              🏛️ 综合裁决
         ✅ 可建仓 / ⚠️ 等待 / ❌ 不通过
```

## 裁决标准

| 裁决 | 条件 |
|------|------|
| ✅ 通过 — 可建仓 | 4维中≥3个 pass，无一 fail，加权分≥70 |
| ⚠️ 有条件通过 | ≥2个 pass，无致命 fail，但时机/估值有瑕疵 |
| ❌ 不通过 | ≥2个 fail，或存在无法反驳的致命攻击 |

## 输出示例

```
📊 中天科技(600522) AI辩论报告

🔬 技术壁垒: 78分 ✅  卡脖子位置明确，但竞争加剧
📊 财务基本面: 72分 ✅  现金流健康，应收略高
💰 估值:          55分 ⚠️  年内涨150%，透支短期预期
⚡ 催化剂:        65分 ✅  15亿大单落地，但已Price In

综合: 68分 ⚠️ 有条件通过
建议: 等回调至40元附近再分批建仓
```
