# 🚦 红绿灯过滤 — 不搞一刀切，扣分累计制

## 原则
- 单一红灯不排除，累计≥3个红灯才排除
- 每个红灯可被反向绿灯抵消（1绿灯=抵消1红灯）
- 目的：天娱数科(4红灯0绿灯)排除，拓维信息(2红灯1绿灯)降权但不排除

---

## 🔴 红灯（每个-1，扣分项）

### 资金面
| 条件 | 扣分 | 可被什么抵消 |
|------|------|------------|
| 龙虎榜买方全是散户席位 | -1 | 机构+游资共振(+1) |
| 机构单笔<5000万 | -1 | 多机构同时买(≥2家)+北向共振(+1) |
| 主力净流出>5亿 | -1 | — |

### 技术面
| 条件 | 扣分 | 可被什么抵消 |
|------|------|------------|
| 连板≥4 + 换手>18% | -1 | 锁仓型游资(炒股养家/欢乐海岸)在场(+1) |
| RSI>85 | -1 | 主线龙头+营收增速>50%(+1) |
| 放量滞涨(量比>2+涨幅<2%) | -1 | — |
| 开板>5次 | -1 | 封单占流通>1%(+1) |

### 基本面
| 条件 | 扣分 | 可被什么抵消 |
|------|------|------------|
| 营收增速<0 | -1 | 扭亏为盈(净利由负转正)(+1) |
| 公司公告澄清/否认概念 | -1 | 有其他独立催化剂(政策/订单)(+1) |
| PE>150x + 营收增速<20% | -1 | PEG<1.5(+1) |
| 大股东/国家队减持>1% | -1 | 机构同期加仓(+1) |

### 宏观
| 条件 | 扣分 | 可被什么抵消 |
|------|------|------------|
| 市场情绪=退潮 | -1 | 唯一性≥90(+1) |

---

## 🟢 绿灯（每个+1，加分项=抵消红灯）

| 条件 | 加分 |
|------|------|
| 唯一性≥90(A股无替代) | +1 |
| PEG<1 | +1 |
| 3家以上机构同时净买 | +1 |
| Q1净利增速>100% | +1 |
| 主线板块龙头(板块涨停≥5家) | +1 |
| 北向+机构+游资三方共振 | +1 |
| 政策刚出+标的直接受益 | +1 |

---

## 📊 判定规则

```
净红灯数 = 红灯数 - 绿灯数

净红灯 ≤0  → 🟢 正常评分，全仓位
净红灯 =1   → 🟡 降权(仓位减半)，仍可推荐
净红灯 =2   → 🟠 观察(仓位≤1成)，标注风险
净红灯 ≥3   → 🔴 排除，不推荐
```

---

## 🔬 案例验证

### 天娱数科(6/10)
```
红灯: 出货形态(-1)+RSI81(-1)+澄清(-1)+假机构(-1)=4
绿灯: 北向买(+1)+AI政策(+1)=2
净红灯: 4-2=2 → 🟠 观察仓位
```
> 之前判定: 🔴排除→ 偏严。修正后: 🟠观察≤1成，仍标注风险。

### 金安国纪(6/10)
```
红灯: RSI91(-1)+开板8次(-1)+公司警告(-1)+PE107(-1)=4
绿灯: 紫阳三方共振(+1)+Q1+764%(+1)=2
净红灯: 4-2=2 → 🟠 观察仓位
```

### 拓维信息(6/10)
```
红灯: 营收-4.7%(-1)=1
绿灯: 华为催化(+1)+主力7.2亿(+1)=2
净红灯: 1-2=-1 → 🟢 正常评分
```
> 之前判定: ❌排除→ 过严。修正后: 🟢正常评分，营收下滑被华为催化和资金面抵消。

### 中钨高新(6/10)
```
红灯: 0
绿灯: 唯一性(+1)+PEG<1(+1)+Q1+264%(+1)=3
净红灯: -3 → 🟢 强烈推荐
```

---

## 生效逻辑

所有标的先过红绿灯 → 净红灯≥3的直接不进入评分 → 进入评分的按净红灯数调仓
